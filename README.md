## dart环境搭建（macos系统）
```
brew tap dart-lang/dart
brew install dart
```
查看dart版本，验证安装完成
```
dart --version
```
编辑器选择 `vscode` ，`vscode` 插件选择`Dart` 和  `Code Runner`。

如果您的系统是 `window` ，可以去 [Dart](https://dart.dev/get-dart) 官网查看具体步骤。
