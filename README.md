## dart 环境搭建（macos 系统）

```
brew tap dart-lang/dart
brew install dart
```

查看 dart 版本，验证安装完成

```
dart --version
```

编辑器选择 `vscode` ，`vscode` 插件选择`Dart` 和 `Code Runner`。
